列表、信噪比与知识管理

发表于 2015-02-07 | | 阅读次数

最近有两个产品比较火，一个是豆瓣2.0新上线，再一个是一个全新的产品「轻单」。两个产品体量不同、目标用户不同、内容不同，但是在产品形态上都一致选择了「列表」这一种组织形式。说起来「列表」并不是一个全新的组织形态，排行榜、收藏夹、相册全部都是形式各样的列表类产品，在互联网这片海洋，沉浮数载，却又在今天有了爆发新生命的迹象。

互联网产品近几年一直站在风口浪尖上，无论是大学毕业生的就业选择还是互联网行业的薪金指标都说明了其极度繁荣甚至有点过度繁荣的现象。作为一个产品的极度爱好者，我并不喜欢这样现象：人人谈产品，可人人都谈不到产品。在这样一个浮躁的环境氛围中，阿北一直是我钦佩的产品人，无论是对豆瓣的打磨，还是个人对产品的思考，都很值得我认真的学习。（有关阿北的产品观可以参考知乎的这个问题：豆瓣阿北在创新工场 Mentoring Session 中分享了哪些做产品及社区的经验？）所以我也希望借这次豆瓣产品改革，谈一谈我的看法。
