极简主义生活实践

发表于 2014-10-28   |     |   热度

首先我们要明确一件事，那就是我们选择任何生活方式都是为了改变现状。那么选择极简主义生活方式可以改变什么现状呢？我认为大体有两种：

  1. 很多经常不用的东西占据生活的大空间;
  2. 大部分的时间并不是在向人生的主要目标努力.

在知乎关于极简主义生活里的答案里有这么一句话，我觉得可以很好地概括:

极简主义就是要通过砍掉生活中一些东西，来减少这些明显弊大于利的事情，从而改善你的生活。
有关极简主义这篇文章写得很棒，我就是从这篇文章了解到这种生活方式的，大家有时间可以去看一看：极简主义生活方式是什么？
至于极简主义生活可以带来什么，我认为有两个：

  1. 在金钱、时间一定的情况下提高你的生活品质
  2. 让你专注于你的主线剧情

哪些地方不够极简？

具体到我有哪些地方不够极简，我整理了一下，大概是这样：

手机

第一个问题就是应用数量过多。我的手机应用大概能到150个左右，这还是在「精挑细选」的情况下保留下来的精华。因为这些应用，手机的速度变慢，经常找不到自己需要的应用，而且占用大量的存储空间，16G的丐版经常不够用。

另一个问题是手机上充斥着社交气体。人人、QQ、微博、微信、非死不可、推鸟……每天80%的消息推送来自这些社交应用。其实大部分的内容都是不重要、我不需要回复或者我不感兴趣的。

最后是碎片信息过载。现在几乎每个应用都在产生待阅读碎片信息，各类新闻应用、微博人人、浏览器、微信公众号、博客……我经常会在准备工作、学习的时间「一不小心」打开这些应用看这些碎片文章，从而浪费了大把大把的整块时间。

电脑

电脑的问题与手机有相似之处，便是软件过多，占用硬盘不说，还会造成选择恐惧症以及效率低下。

另外电脑的文件管理体系过于杂乱，文件总是乱放，使得无论是存储文件还是查找文件都异常困难。

最后一个小问题就是桌面堆了太多的东西，既影响心情，又影响效率。

穿衣

有很多人可能奇怪，衣服和极简主义有什么关系，穿衣服又能遇到什么事呢？事实上，我还就真有一些小麻烦：

一个是在穿衣搭配上总是耗费太长的时间，每天早上都要花一段时间犹豫今天穿什么这个「终极命题」，最后穿出去的可能也不是自己满意的。

再一个问题就是有些衣服的质量不够高，经常穿不了两季就不怎么穿了。

如何改变？

极简主义生活的关键在于减法，减掉不必要的东西，保留重要的，就是改变的最高原则：

手机

针对手机应用过多的问题，可以花一段时间记录每一个应用的使用次数和使用时长。如果在半个月内都几乎没有使用的应用就可以安心卸载了。有些特定的时间需要的应用（比如旅游类、订火车票等）可以在需要的时候再下载下来。

社交过多的问题也可以拼命做减法。卸载掉QQ、人人、推鸟等几乎没有紧急事情的应用。关掉除短信、微信外的所有社交应用推送通知，把不需要关注的群全部取消通知。还有一个小窍门：关闭微信朋友圈的红点提醒（我-设置新消息通知-朋友圈照片更新）。

而处理碎片信息过载，则要把减法和系统控制结合起来，具体操作方法如下：

  1. 订阅内容全部自动存入RSS应用（我用的是「feedly」），但只做筛选，不阅读
  2. 非订阅内容需要阅读的以及「RSS应用」里面值得阅读的订阅内容放入稍后阅读应用「pocket」
  3. 待到碎片时间阅读pocket，读过的标记「已读」、「精华」等标签
  4. 读过至少两遍的内容如果认为值得存档则存储到evenote中
电脑

电脑端的减法则要轻松不少，主要是文件系统的整理。建立一个逻辑清晰，层次分明的文件归档系统是最重要的事情。

至于软件过多、桌面拥挤：删除就是了。

穿衣

针对穿衣搭配花费过多时间的问题，可以精简衣服的种类和数量，保证够穿、耐穿就可以。组成几个固定的搭配，每天晚上睡觉前把第二天要穿的衣服整理好。

至于衣服太多，质量普遍不高的问题，采用减法，扔掉不常穿的衣服，够穿是上限。同时以后够买衣服尽量买质量好，材质好的衣服。

每周定期洗衣服，内衣之类的每天洗，养成习惯。

当下实践成果

现在的我在手机、电脑两个方面已经基本按照上面说的方式去执行，感觉真的非常好，尤其是手机清静后带来的效率提升是难以形容的。
但是在很多其他方面例如饮食、宿舍的整理、穿衣等还是和极简生活有很大的差距，今后准备继续努力。比如这周三就准备彻底把宿舍「极简」一下。

最后的话

其实极简生活方式并不比其他的生活方式更优越，只是在不同人生阶段、价值观下的不同选择罢了。我选择践行极简主义生活方式的原因是我有着明确的个人长远目标：事业与家庭，我希望这两个主要目标之外的事物可以尽可能少占用一些我的时间、精力、金钱。
