「计算广告」章节思考题整理

发表于 2016-06-05 | | 阅读次数 |
最近仔细研读了刘鹏老师的「计算广告」这本书，对于广告产品有了一个初步系统的认识。这本书每一章节后的延伸思考题是一大特色，题目都比较开放，没有确切的答案，对于自我学习是一个很好的检测方式。

最近仔细研读了刘鹏老师的「计算广告」这本书，对于广告产品有了一个初步系统的认识。这本书每一章节后的延伸思考题是一大特色，题目都比较开放，没有确切的答案，对于自我学习是一个很好的检测方式。书中的习题有很大一部分我目前还给不出答案，仅仅有不到一半的习题（因为后面技术部分的习题我压根都没敢看）我有自己初步的答案，贴在这里也算是一个学业的小结，未来还是要深入地思考这些习题，并且要多找有经验的老师多多请教。

题目1

CPA/ROI 的结算方式看似对广告主有利，实际也催生了一些变形的推广方式。请考察你接触到的 CPA/ROI 渠道，并研究其与 CPM 或 CPC 渠道的关键差异。

-CPA/ROI渠道：移动应用下载（预装、捆绑）、淘宝客、垂直广告网络等

-与CPM、CPC 渠道的关键差异：广告监测的对象不一样

题目2

对广告产品而言，优化利润与优化 ROI 有什么区别？

-广告活动的利润：T 次广告展示中总收入与总成本的差

-广告活动的 ROI：T 次广告展示活动中总收入与总成本的比

-当成本一定时，两者是相同的，但是当成本不一定时两者有区别：区别是什么？

题目3

如果向需求方提供 API 来代替一般的 UI 投放方式，会对广告产品的运营产生什么样的影响。

-投放系统压力增大

-提升大规模投放广告主、代理的投放效率

题目5

搜索广告南区的产品策略在不同搜索引擎中差别很大，请从用户产品的角度出发，谈谈你认为合理的策略。

-策略分类：1、照搬北区广告 2、照搬东区广告

-不同策略的特点：

照搬北区广告——保证相关性，价格降低，

照搬东区广告——相关性无法保证，类型泛化，竞价保证

题目6

在网址导航类产品的销售中，往往采取按月线下竞价然后签订广告位合约的模式，这属于竞价广告还是合约广告。

-还算是合约广告

-竞价广告的本质是将量的约束从交易过程中去除，每一次广告都要竞价

题目7

重定向的广告效果往往显著高于其他定向方式，不过这部分是由于其人群选择的特殊性，应如何公允地评价一次重定向广告的实际增量效果？

-重定向：某个广告主过去一段时间的访客

-特殊性：很有可能是老客户，那么对于拉新的效果衡量不好（即回头客是否由广告带动）

-方案1：A\B Test -将老客户分为两组，一组投放广告，一组不投放，观测回头率区别。

-方案2：细化老客户标签，分为已转化和未转化两组，已转化的不投放，未转化的投放。

题目8

对于大型广告主和中小广告主，移动互联网营销相比 PC 互联网营销来说，发生了什么有利或不利的变化？

-它最本质的出发点就是从人出发，而不是从销售出发。每个人每时每刻都离不开手机。

题目9

移动设备与 PC 的跨屏营销主要需求场景有哪些？
